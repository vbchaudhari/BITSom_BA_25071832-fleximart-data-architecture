# Part 3: Data Warehouse & Analytics

## Overview
In this part, I designed a Star Schema Data Warehouse to help FlexiMart analyze historical sales data efficiently.

## Files Included
* **star_schema_design.md**: Documentation of the Fact and Dimension tables.
* **warehouse_schema.sql**: SQL code to create the Data Warehouse tables.
* **warehouse_data.sql**: SQL code to insert dummy data for testing.
* **analytics_queries.sql**: Complex OLAP queries for business intelligence reports.
